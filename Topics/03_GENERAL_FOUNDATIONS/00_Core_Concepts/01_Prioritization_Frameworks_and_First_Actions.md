# General Foundations 01: Prioritization Frameworks and First Actions

Date: March 20, 2026

## Core Frameworks Used
1. ABCs (airway, breathing, circulation)
2. Immediate safety threats
3. Acute change over chronic/stable findings
4. First action vs best overall plan distinction

## Rules Applied Repeatedly
- Stabilize first, report second, document after urgent actions begin.
- If oxygenation is compromised, intervene immediately and reassess quickly.
- Partial improvement with ongoing distress still requires escalation.
- New confusion + hypotension/cool clammy skin indicates urgent perfusion concern.

## High-Yield Scenario Outcomes
- SpO2 88% + RR 30 + restlessness: prioritize first.
- First action: oxygen + head-of-bed elevation + reassessment.
- Next action if persistent distress: urgent SBAR escalation.
- Suspected DVT: bed rest, avoid massage/ambulation, notify provider.
- New confusion attempting to climb out of bed: immediate safety priority.

## Common Trap Patterns
- Calling provider before immediate stabilization.
- Delaying escalation after only minor response.
- Choosing paperwork/reporting before patient stabilization.
