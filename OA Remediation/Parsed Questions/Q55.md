# Question 55

**Prompt:** A nurse is disposing of a trach suction catheter in a client's room. Which infection control action is correct?
**Type:** Single choice.

## Options
1. Dispose of catheter in regular trash
2. Wrap it in gloves and place in biohazard bag
3. Place in red biohazard bin using sterile gloves
4. Rinse and discard in sharps container
